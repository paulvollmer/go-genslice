# go-genslice

`go-genslice` is a cli tool to generate go slice code.

![CI][ci]
[![GoDev](https://img.shields.io/static/v1?label=godev&message=reference&color=00add8)][godev]

## Install

```
go get -u github.com/paulvollmer/go-genslice
```
